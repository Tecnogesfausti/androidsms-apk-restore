Expected working registration flow
1. Install the APK on an Android phone with Google Play Services
2. Open the app
3. Go to Home > Start
4. Use Sign Up with the Base URL and Private token above
5. After successful registration, open Cloud settings and read the FCM token

If registration still fails
- Check that the phone has internet access
- Check that the app is using:
  http://smsgateway.mihuerto.uk/api/mobile/v1
- Check that the private token matches exactly
- Review logcat for Firebase and registration errors:
  adb logcat | grep -i -E 'firebase|fcm|gateway|retrofit|pushservice'
- If the message queue does not move while FCM is flaky:
  switch the app to SSE_ONLY and re-arm ONLINE
